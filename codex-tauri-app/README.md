# Codex Switch Desktop

English:

This is the Tauri desktop app for Codex Switch. It provides a local GUI for saving, switching, and checking Codex account configurations.

Quick start:

```bash
npm install
npm run tauri dev
```

Build:

```bash
npm run tauri build
```

The packaged app will be generated under `src-tauri/target/release/bundle/`, including the macOS `.dmg` output.

中文：

这是 Codex Switch 的 Tauri 桌面应用，提供本地图形界面，用于保存、切换和查看 Codex 账号配置。

快速开始：

```bash
npm install
npm run tauri dev
```

构建：

```bash
npm run tauri build
```

构建后的安装包位于 `src-tauri/target/release/bundle/`，其中 macOS 会生成 `.dmg`。
